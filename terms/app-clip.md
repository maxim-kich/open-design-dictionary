# App Clip

## Definition
Lightweight iOS app experience no download.

## Examples
- Starbucks App Clip order
- Parking payment Clip
- Event ticket Clip
- 50MB size limit

## Do's and Don'ts

**Do's**
- Single focused function
- Core ML integration
- Passkeys Sign in Apple
- AR Quick Look support

**Don'ts**
- Full app experience
- Multiple features
- Large download size
- No Apple integration
