# Build-Measure-Learn

## Definition
MVP feedback loop continuous product learning.

## Examples
- Landing page MVP
- 100 signups 7-day waitlist
- Pivot to paid feature
- 3-day BML cycles

## Do's and Don'ts

**Do's**
- Smallest viable test
- Quantitative + qualitative
- Pre-defined success
- Rapid iteration

**Don'ts**
- Big bang launch
- Qualitative feedback only
- No success criteria
- Monthly learning cycles
