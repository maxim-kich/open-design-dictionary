# Contribution Review Process

## Definition
Peer review merge standards.

## Examples
- Design system PR template
- Automated linting tests
- 2 approver minimum
- Changelog automation

## Do's and Don'ts

**Do's**
- Clear acceptance criteria
- Automated checks first
- Cross-team review
- Comprehensive testing

**Don'ts**
- Direct main merge
- Single reviewer
- No automated checks
- Verbal approval
