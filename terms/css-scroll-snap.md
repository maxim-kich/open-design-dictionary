# CSS Scroll Snap

## Definition
Snaps scroll container to specific points.

## Examples
- Fullscreen sections snap
- Horizontal card carousel
- Mobile snap points
- Touch-friendly snapping

## Do's and Don'ts

**Do's**
- scroll-snap-align
- Smooth snap behavior
- Keyboard accessibility
- Fallback smooth scroll

**Don'ts**
- Jerky snapping
- No momentum scrolling
- Desktop-only snap
- Tiny snap targets
