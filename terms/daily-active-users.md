# Daily Active Users (DAU)

## Definition
Unique users using product per calendar day.

## Examples
- DAU 24.7k MAU 187k
- DAU/MAU stickiness 13.2%
- DAU growth 8% MoM
- Segment DAU by feature

## Do's and Don'ts

**Do's**
- Track DAU/MAU ratio
- Segment by user type
- Monitor day-of-week
- Set growth targets

**Don'ts**
- Report raw DAU only
- No stickiness ratio
- Daily fluctuations ignored
- All-time unique
