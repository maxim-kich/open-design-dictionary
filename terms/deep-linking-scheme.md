# Deep Linking Scheme

## Definition
URL schemes opening specific app screens.

## Examples
- myapp://profile/123
- universal links iOS
- App Links Android
- Deferred deep linking

## Do's and Don'ts

**Do's**
- Clear URL structure
- Fallback web experience
- Test all link types
- Analytics tracking

**Don'ts**
- Obfuscated URL schemes
- No web fallback
- Broken universal links
- Desktop-only linking
