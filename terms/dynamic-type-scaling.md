# Dynamic Type Scaling

## Definition
Text resizes with system accessibility settings.

## Examples
- SF Pro Dynamic Type
- Roboto scalable text sizes
- Preferred font scaling
- Minimum 11pt readable

## Do's and Don'ts

**Do's**
- Honor Dynamic Type
- Test all scaling levels
- Responsive layouts
- Legible at max scale

**Don'ts**
- Fixed font sizes
- Layout breaks at 200%
- Minimum 16px clamp
- Desktop font sizing
