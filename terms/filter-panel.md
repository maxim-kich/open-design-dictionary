# Filter Panel

## Definition
Collapsible panel containing facet controls.

## Examples
- Left sidebar on desktop
- Modal overlay on mobile
- Top filter chips
- Drawer from top/bottom

## Do's and Don'ts

**Do's**
- Sticky on scroll
- Visual count badges
- Collapsible sections
- Keyboard accessible

**Don'ts**
- Always expanded
- Tiny touch targets
- No clear/reset
- Poor mobile layout
