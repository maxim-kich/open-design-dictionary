# Keyboard Navigation

## Definition
Tab order and key actions enabling non-mouse interface control.

## Examples
- Tab through form fields
- Arrow keys in menus
- Enter to activate buttons
- Escape to close modals

## Do's and Don'ts

**Do's**
- Logical tab order
- Clear focus indicators
- Keyboard shortcuts
- Skip repetitive navigation

**Don'ts**
- No keyboard support
- Confusing tab order
- Hidden focus states
- Trap keyboard focus
