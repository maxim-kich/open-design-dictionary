# Page Load Time

## Definition
Time request to interactive ready.

## Examples
- LCP 2.4s TTI 3.8s
- Mobile 4.2s desktop 1.9s
- Core Web Vitals correlation
- Revenue impact analysis

## Do's and Don'ts

**Do's**
- Real user monitoring
- Business impact correlation
- Platform optimization
- Continuous tracking

**Don'ts**
- Lab synthetic only
- No revenue correlation
- Desktop focus
- Monthly snapshots
