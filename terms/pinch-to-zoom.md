# Pinch to Zoom

## Definition
Two-finger spread/shrink controls magnification.

## Examples
- Photo gallery zoom
- Map pinch navigation
- PDF document zoom
- Canvas drawing zoom

## Do's and Don'ts

**Do's**
- Smooth zoom animation
- Double-tap zoom preset
- Minimum/maximum bounds
- Zoom with pan

**Don'ts**
- Zoom buttons only
- Pixelated zoom quality
- No pan after zoom
- Desktop-only zoom
