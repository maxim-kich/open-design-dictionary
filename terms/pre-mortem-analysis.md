# Pre-mortem Analysis

## Definition
Imagine project failed predict prevent risks.

## Examples
- 'We failed because...'
- 25 risks identified
- Mitigation plans created
- Cross-team workshop

## Do's and Don'ts

**Do's**
- Future perfect tense
- Diverse perspectives
- Actionable mitigations
- RACI assignment

**Don'ts**
- Blame assignment
- Optimism bias only
- No mitigation plans
- Post-failure analysis
