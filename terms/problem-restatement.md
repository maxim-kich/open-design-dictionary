# Problem Restatement

## Definition
Reframe problem multiple ways to unlock new solution angles.

## Examples
- "How might we..." variations
- Opposite framing
- User perspective shift
- Constraint removal

## Do's and Don'ts

**Do's**
- Multiple reframes
- Challenge assumptions
- Include stakeholder views
- Test framings

**Don'ts**
- Single restatement
- Keep original bias
- Skip validation
- Abstract only
