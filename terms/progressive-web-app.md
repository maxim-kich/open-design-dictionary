# Progressive Web App

## Definition
Website with native app features (install offline).

## Examples
- App manifest + service worker
- Offline page support
- Add to home screen
- Push notifications

## Do's and Don'ts

**Do's**
- HTTPS required
- App-like experience
- Fast offline fallback
- Clear install prompts

**Don'ts**
- No service worker
- Poor offline experience
- Desktop-only PWA
- Missing manifest
