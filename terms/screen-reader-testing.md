# Screen Reader Testing

## Definition
Manual testing with NVDA VoiceOver TalkBack.

## Examples
- NVDA Chrome testing
- VoiceOver Safari iOS
- TalkBack Android
- 2hr focused session

## Do's and Don'ts

**Do's**
- Real screen reader users
- Task-based testing
- All platform combinations
- Detailed bug reports

**Don'ts**
- Demo with screen reader
- Quick scan only
- Single platform
- No task context
