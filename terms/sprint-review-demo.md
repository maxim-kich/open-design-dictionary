# Sprint Review Demo

## Definition
Stakeholder showcase working software.

## Examples
- 8 stories completed demo'd
- Stakeholder feedback
- Updated backlog
- 1hr max duration

## Do's and Don'ts

**Do's**
- Demo working software
- Invite all stakeholders
- Real user scenarios
- Document feedback

**Don'ts**
- Slide presentation
- Team-only review
- No stakeholder input
- >2hr duration
