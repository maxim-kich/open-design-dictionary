# Starfish Retrospective

## Definition
Five-category retrospective: keep, more, less, stop, start.

## Examples
- Keep daily standups
- More user testing
- Less meetings
- Stop scope creep
- Start documentation

## Do's and Don'ts

**Do's**
- Equal time per category
- Specific examples
- Vote on priorities
- Create action items

**Don'ts**
- Skip categories
- Vague feedback
- No follow-through
- Blame-focused items
