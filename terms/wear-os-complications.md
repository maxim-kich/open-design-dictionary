# Wear OS Complications

## Definition
Customizable glanceable data watch face.

## Examples
- Weather complication
- Steps fitness tracker
- Calendar next event
- Custom complication API

## Do's and Don'ts

**Do's**
- Glanceable information
- Tap for details
- Battery efficient
- Contextual relevance

**Don'ts**
- Dense information
- Deep interaction required
- Battery drain
- Static data
