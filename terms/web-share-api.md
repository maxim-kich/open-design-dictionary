# Web Share API

## Definition
Native browser sharing interface.

## Examples
- Share button triggers native
- navigator.share() API
- Fallback copy link
- Mobile optimized sharing

## Do's and Don'ts

**Do's**
- Request permission first
- Fallback gracefully
- Mobile-first implementation
- Test all platforms

**Don'ts**
- Assume universal support
- Ugly custom modals
- Desktop-only sharing
- No fallback copy
